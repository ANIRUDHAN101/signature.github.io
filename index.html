Signature Authentication System

Using Deep Learning for Secure and Reliable Signature Verification

Abstract Methodology Results Contact

Abstract

This project develops a signature authentication system using a deep learning approach. EfficientNetV2 is utilized as the shared-weight encoder, transforming signature images into embeddings. A Contrastive Loss function trains the model to minimize embedding distances for similar signatures and maximize them for dissimilar ones. The system uses RGB images as color is irrelevant for signature analysis. During testing, a threshold of 0.00093 determines whether signatures are genuine or forged. The model was trained and evaluated on the Signature Verification Dataset.

Methodology

  • Input images are preprocessed and converted to RGB.
  • EfficientNetV2 is used as an encoder with shared weights for pairwise signature comparisons.
  • Embeddings are passed through the Contrastive Loss function:

Contrastive Loss:
𝓛 = 1/N Σ [(1 − 𝑦) · 𝑑²(x₁, x₂) + 𝑦 · max(0, 𝑚 − 𝑑(x₁, x₂))²]
where 𝑑(x₁, x₂) is the Euclidean distance between embeddings, 𝑦 is the label, and 𝑚 is the margin.

Results

The model achieved effective results in distinguishing genuine from forged signatures. Testing used a contrastive distance threshold of 0.00093:

  • 𝑑(x₁, x₂) < 0.00093: Genuine signatures
  • 𝑑(x₁, x₂) ≥ 0.00093: Forged signatures
