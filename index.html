About the Project

This project focuses on developing a robust AI-based system for verifying handwritten signatures to prevent fraud and enhance authentication systems.

Methodology

We use deep learning models combined with preprocessing techniques to analyze and verify signatures.

Key techniques include:

  • Image preprocessing and feature extraction
  • Convolutional Neural Networks (CNNs)
  • Similarity metrics such as cosine similarity

Results

The system achieves high accuracy in distinguishing between genuine and forged signatures.
